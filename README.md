# 網⾴設計 or 介⾯設計？
### 資訊架構 Information Architecture (IA)
包括网站、App的结构安排以及内容是如何组织的，目的是帮助用户快速找到信息并且完成操作。换句话说，信息架构就是组织好网站或者App的结构，让用户明白目前所在的位置，知道所找的信息在哪里。信息架构师的工作包括设计网站地图、层次结构、分类、导航和元数据。当内容规划师开始将内容分开并归类的时候，这就是信息架构。
